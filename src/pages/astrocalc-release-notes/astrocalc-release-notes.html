AstroCalc Release Notes

Version 1.1.0 on 2019-10-17

  • Auto-store. Offers the option to automatically save the accumulator to the specified memory location. This option is available in the following commands: Observatory, Universal Time, Local Time, Coordinates, Precession, and Transformation.
  • Remembers the last values entered in a dialog. Saves time by eliminating the need to re-enter the many values that comprise the Observatory, Coordinates, and Universal Time dialogs.
  • Now button added to Universal Time dialog in case you want to quickly override the last values with the current time.
